This document causes bd-smoke to load scripts given in the URL query string and then execute smoke.runDefault() without AMD.

Make sure you specify something to load in the URL query string! Here is an example that loads the test ./smoke/test/minimal-example:

http://localhost:8080/altoviso/bd-smoke/browser-runner.html?p=./test/minimal-example.js

Here is a helper that loads the minimal example test as an example: ./browser-runner.html?p=./test/minimal-example.js

Open the console to see the results.
